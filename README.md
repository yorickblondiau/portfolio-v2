Description du projet

Ce projet fait suite à la **Commande 1 – Portfolio V1**.  
L’objectif est de créer un **site web accessible et responsive** en HTML et CSS, avec:
-	un **mode clair/sombre** (accessibilité visuelle)
-	une **grille fluide** respectant la mise en page du PSD
-	une **galerie Lightbox CSS**
-	des **animations sobres** + **balise vidéo** exportée d’After Effects (.mov / .mp4)
-	un **effet de parallaxe CSS**
-	un **formulaire de contact HTML5**
-	une **mise en ligne** sur **Netlify** ou **GitHub Pages**
